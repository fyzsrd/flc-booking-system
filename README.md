# FLC Booking System

This project is a Java-based software system developed for managing group exercise lesson bookings at Furzefield Leisure Centre (FLC).

## Features

* View timetable by day or exercise type
* Book exercise lessons (Yoga, Zumba, etc.)
* Change bookings (if spaces are available)
* Prevent time conflicts
* Add reviews and ratings (1–5)
* Generate reports:

  * Number of members per lesson + average rating
  * Highest income generating exercise

## Technologies Used

* Java
* JUnit (for testing)
* Git (version control)

## Project Structure

* `src/` → main source code
* `test/` → unit tests

## Purpose

This project is developed as part of the **Programming for Software Engineers (7COM1025)** coursework.
